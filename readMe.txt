Die Applikation lässt sich über das Terminal mit npm start öffnen, wenn man sich im Ordner fontastic befindet.
Alternativ kann man sie über folgenden Link aufrufen:

p3-team7.vercel.app

Anstelle des echten Artefakts kann das beigefügte Bild des als Marker verwendet werden.
Quelle: https://www.flickr.com/photos/32449131@N07/6175504211/

Die AR Funktion lässt sich über jeden AR Knopf erreichen, funktioniert jedoch exemplarisch nur für das Artefakt "dem lebendigen Geist".

Am besten funktioniert unser Prototyp im Landscape Format.
